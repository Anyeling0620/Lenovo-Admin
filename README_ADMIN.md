快速说明 — 中后台商城管理系统（示例框架）

依赖安装（使用 pnpm）：

```bash
pnpm add antd @ant-design/icons
```

或使用 npm：

```bash
npm install antd @ant-design/icons
```

运行项目：

```bash
pnpm install
pnpm dev
```

说明：
- 已实现基于 `antd` 的 `MainLayout`（侧边栏、页头、面包屑、内容区）。
- 示例页面位于 `src/pages/`。
- 侧边菜单封装在 `src/layouts/SidebarMenu.tsx`。
